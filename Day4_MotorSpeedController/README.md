## 📌 Day 4 – PCB Design Challenge

### 🔧 Project: Motor Speed Control Circuit

### 🧩 Components Used

* NE555P Timer IC
* TIP122 Transistor
* Diode (1N4007)
* Capacitor
* Resistor
* LED
* Screw Terminals

### ⚙️ Key Functionalities

* Generates PWM signal for motor speed variation
* Adjustable motor speed control
* TIP122 handles higher motor current
* LED indicates power/status
* Protection diode prevents back EMF damage

### 📘 Project Description

This project focuses on designing a motor speed control PCB using the NE555 timer IC for PWM generation. By adjusting the control input, the duty cycle changes, allowing smooth variation of DC motor speed. The TIP122 transistor acts as the driving stage for handling motor load efficiently. This project helped in understanding PWM-based motor control and practical PCB routing techniques.

### 🌍 Applications

* DC motor speed control
* Robotics projects
* Automation systems
* Mini cooling fans and DIY electronics

### 🚀 Future Improvements

* Add microcontroller-based smart control
* Include display for RPM monitoring
* MOSFET-based high-efficiency driver stage
* Compact SMD PCB version
