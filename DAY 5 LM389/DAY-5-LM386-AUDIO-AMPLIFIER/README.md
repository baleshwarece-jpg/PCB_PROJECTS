📁 Day 5 – LM386 Audio Amplifier PCB Design
🔊 Project Overview

This project focuses on designing an LM386 based Audio Amplifier PCB capable of amplifying weak audio signals into audible output through a speaker. The PCB was designed with proper component placement, routing, grounding, and filtering techniques to achieve stable and low-noise audio amplification.

🧩 Components Used
LM386 Audio Amplifier IC
Resistors
Capacitors
Potentiometer
LED Indicator
Audio Input Connector
Speaker Output Terminal
DC Power Connector
⚙️ Key Functionalities
Amplifies low-level audio signals
Adjustable volume control using potentiometer
Compact and efficient low-power amplifier
Capacitor filtering for stable audio output
LED indication for power status
Easy speaker interfacing through output terminal
📘 Project Description

Designed the complete schematic and PCB layout for an LM386 audio amplifier circuit using PCB design software. The project involved:

Schematic creation
Component footprint assignment
PCB routing and placement
Power and signal line optimization
Gerber file generation for fabrication

Special attention was given to reducing noise and maintaining clean signal paths for better audio performance.
