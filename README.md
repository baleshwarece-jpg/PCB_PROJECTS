# 30 Days PCB Design Challenge

A daily PCB design challenge — one new circuit every day, from power supplies to motor control to audio amplifiers.  
Designed in KiCad | Documented daily | Built for learning and real-world hardware practice.

---

## About This Repository

This repository documents my 30-day PCB design journey. Every day I design, route, and document a new circuit board — starting from the basics and progressively working toward more complex, real-world hardware. Each project folder is self-contained and includes the schematic, PCB layout, and project files.

The purpose is straightforward: build consistency, develop practical PCB design skills, and create a portfolio of hardware projects that reflect real engineering work — not just tutorials.

### Day 5 — LM386 Audio Amplifier PCB
[`DAY-5-LM389`](./DAY-5-LM389)

**Components Used**
- LM386 IC, Capacitors, Resistors, Potentiometer, LED, Audio Connectors, Speaker Terminal

**What It Does**
A low-voltage audio power amplifier built around the LM386 IC. Takes a weak audio signal at the input and drives a small speaker at the output. Volume is adjusted via a potentiometer on the input. Bypass capacitors reduce power supply noise that would otherwise appear as hum in the audio output. The LM386 can deliver up to 200 mW into an 8-ohm speaker load.

This project introduced working with mixed signal and power routing on the same board, and the importance of keeping audio signal traces away from power traces to minimize interference.

### Day 7 — 3.3V LDO Regulator Board
[`Day7-3V3-LDO-Regulator`](./Day7-3V3-LDO-Regulator)

**Components Used**
- AMS1117-3.3 LDO IC (SOT-223), 100µF electrolytic capacitors (x2), 10µF MLCC ceramic capacitors (x2), 1kΩ resistor, LED, 2-pin input/output connectors

**What It Does**
Accepts an unregulated DC input between 4V and 12V and outputs a clean, stable 3.3V at up to 800 mA — the standard supply voltage for most modern microcontrollers including the ESP32, STM32, and nRF52 families. Input and output capacitor pairs handle both bulk energy storage and high-frequency decoupling. The SOT-223 package's exposed thermal tab is connected to the output net and requires a copper pour beneath it on the PCB for adequate heat dissipation under sustained load.

This project reinforced how much physical component placement on the PCB matters — the ceramic caps must sit as close as possible to the IC pins to be effective at filtering high-frequency noise.
